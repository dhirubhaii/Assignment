Bootstrap Basics

Bootstrap helps in building responsive websites quickly and easily with pre-built CSS and JS components.

What is Bootstrap?

Bootstrap is a popular open-source front-end framework that includes CSS and JavaScript components you can use for responsive, mobile-first web projects.

Key features: grid system, buttons, forms, typography, navigation, modals, tooltips, accordions, and more.

Responsive Grid and Buttons

The grid system allows you to create flexible page layouts that adjust to different screen sizes. Columns can resize, combine, or stack depending on the device. Pre-styled buttons like btn-success ensure a consistent look with hover, active, and focus states.

Bootstrap Cards

Cards are flexible containers for images, text, and actions. They are widely used for feature blocks, blog posts, and product listings.

  • Card with Image: Combines image, title, text, and button.
  • Card without Image: Works without media and can use border styles for emphasis.
  • Light Background Card: Uses contextual classes like bg-light for highlighting.

Bootstrap Modal

Modals are overlays that display content without leaving the page. They are useful for dialogs, forms, or notifications. Bootstrap handles focus trapping and keyboard accessibility automatically.

Bootstrap Modal
This is a modal example. It overlays content with focus trapping and accessibility support.

Bootstrap Tooltip

Tooltips display small info boxes when hovering or focusing on an element. They’re useful for extra context, explaining icons, or clarifying form fields.

Bootstrap Accordion

Accordions let you toggle the visibility of content panels. They’re ideal for FAQs, previews, or multi-step guides. Only one section is expanded at a time for better readability.
