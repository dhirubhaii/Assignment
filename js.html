JavaScript Overview

JavaScript is a versatile programming language used to add interactivity, logic, and dynamic behavior to webpages. Unlike HTML (structure) and CSS (style), JavaScript controls the behavior of a webpage.

Introduction to JavaScript

Definition: JavaScript is a programming language that makes webpages interactive and dynamic.

  • Responds to user actions (clicks, input)
  • Updates content dynamically
  • Validates forms
  • Builds games, apps, and web tools
console.log("Hello, JavaScript!");
alert("Welcome to my JavaScript page!");

JavaScript Basics (Quick View)

Variables & Data Types:

Variables: let, const (avoid var in modern JS).

Data Types: Number, String, Boolean, Null, Undefined, Object, Array.

let name = "Dhiraj";
const age = 20;
let isStudent = true;
console.log(name, age, isStudent);
Operators:

Arithmetic: + - * / %

Comparison: ==, ===, !=, !==, >, <,>=, <=< /p>

Logical: &&, ||, !

let result = (5 + 3) * 2;
console.log(result); // 16
console.log(age > 18 && isStudent); // true
Functions:

Definition: Reusable blocks of code that perform tasks.

function greet(userName) {
  return `Hello, ${userName}!`;
}
console.log(greet("Dhiraj"));
Conditional Statements:

Decide code flow using if, else if, else, switch. 

let hour = 10;
if(hour < 12){
  console.log("Good Morning!");
} else if(hour < 18){
  console.log("Good Afternoon!");
} else {
  console.log("Good Evening!");
}
Loops:

Repeat tasks with for, while, do...while, or forEach.

for(let i = 1; i <= 5; i++){
  console.log("Count: " + i);
}
Events:

Definition: JS listens and responds to user interactions like clicks or typing.

document.querySelector("#js-btn")
.addEventListener("click", () => {
  document.querySelector("#js-msg").textContent = "Button clicked!";
});
Arrays & Objects:

Arrays: Store multiple values.

Objects: Store data in key-value pairs.

let fruits = ["Apple", "Banana", "Mango"];
let person = {name: "Dhiraj", age: 20};
console.log(fruits[1]); // Banana
console.log(person.name); // Dhiraj

DOM Manipulation

The DOM (Document Object Model) represents the structure of your webpage as a tree of elements. JavaScript can access and change these elements dynamically, letting you update text, add new elements, or even remove parts of the page without reloading.

Example: Adding a new paragraph to the page with JavaScript:

let para = document.createElement("p");
para.textContent = "This is dynamically added by JS!";
document.body.appendChild(para);
  • document.createElement("p") → creates a new paragraph element.
  • para.textContent → sets the text inside the element.
  • document.body.appendChild(para) → inserts it into the page.

This is how JavaScript makes webpages interactive and alive.
