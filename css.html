CSS Fundamentals & Demos

Learn how CSS styles your web page with examples you can see live.

1. CSS Inclusion Methods

<!-- Inline CSS -->
<p style="color: red;">This is inline CSS</p>

<!-- Internal CSS -->
<style>
  .internal-demo { color: blue; font-weight: bold; }
</style>
<div class="internal-demo">This text uses internal CSS.</div>

<!-- External CSS -->
<link rel="stylesheet" href="styles.css">
<div class="external-demo">This text uses external CSS</div>

This text is styled with inline CSS.

2. CSS Selectors

<!-- Class Selector -->
<p class="class-demo">This text uses a class selector</p>

<!-- ID Selector -->
<p id="id-demo">This text uses an ID selector</p>

<!-- Tag Selector -->
<p>This text uses a tag selector</p>

Selectors target HTML elements to apply styles. Common types are class, ID, and tag selectors.

3. Box Model

<div class="box-model-demo">Box Model Example</div>

<!-- CSS -->

  padding: 20px;
  border: 2px solid black;
  margin: 10px;
}

Every element in CSS is a rectangular box, composed of content, padding, border, and margin.

6. Flexbox Layout

<div class="flex-demo">
  <div class="flex-item">Flex 1</div>
  <div class="flex-item">Flex 2</div>
  <div class="flex-item">Flex 3</div>
</div>

<!-- CSS -->
.flex-demo { display: flex; gap: 10px; }
.flex-item { padding: 20px; background: #e9c46a; }

Flexbox is a layout model that allows elements to align and distribute space efficiently within a container.

7. CSS Transitions & Animations

<button class="hover-btn">Hover Me!</button>

<!-- CSS -->
.hover-btn {
  transition: background 0.3s;
}
.hover-btn:hover {
  background: #2a9d8f;
}

CSS transitions animate changes to properties, and keyframes allow complex animations without JavaScript.

8. CSS Grid Layout

<div class="grid-demo">
  <div class="grid-item">1</div>
  <div class="grid-item">2</div>
  <div class="grid-item">3</div>
  <div class="grid-item">4</div>
</div>

<!-- CSS -->
.grid-demo { display: grid; grid-template-columns: repeat(2, 1fr); gap: 10px; }
.grid-item { padding: 20px; background: #f4a261; }

Grid layout allows you to arrange elements in rows and columns with precise control over spacing and alignment.

9. CSS-Only Effects

<div class="hover-box">Hover Me</div>

<!-- CSS Hover -->
.hover-box:hover { background: #2a9d8f; color: white; }

<!-- Keyframe Animation -->
.animation-demo { animation: bounce 2s infinite; }

@keyframes bounce { 0%,100% { transform: translateY(0); } 50% { transform: translateY(-20px); } }

<!-- Pseudo-element -->
.pseudo-demo::after { content: ' - after text'; }
