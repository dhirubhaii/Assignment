What is HTML?

HTML (HyperText Markup Language) is the backbone of all web pages.

Demo: Basic HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Hello, World!

This is a paragraph.

Forms & Inputs

Forms collect user input. Common elements include <input>, <textarea>, and <button>. They are essential for contact forms, surveys, and login pages. 

<form>
  <input type="text" placeholder="Name">
  <input type="email" placeholder="Email">
  <textarea></textarea>
  <button>Submit</button>
</form>
